Página de ventas DentalCare

Una página web desarrollada con el fin de la realización de ventas en línea de productos de higiene y cuidado bocal, mediante un carrito de compras el cliente podrá realizar el pedido y un usuario administrador se podrá hacer cargo de administrar y supervisar dicho pedido ademas de los productos y categorías existentes, se utilizó HTML y CSS para la creación de un front-end y se implementó el modelo vista controlador en PHP junto con una base de datos MySQL para el desarrollo del back-end.


Blog en JavaScript

Botser

Botser es un bot creado a partir del uso de Node JS y la librería de Discord que dispone con múltiples funciones que pueden ser agregadas, para este bot se ha implementado un par de funciones básicas como lo son un saludo hacia cualquier usuario por medio del comando "hola" así como una despedida por otro comando que es "adios", además de permitir llamar una imagen con otro comando el cual es "!imagen", por cada uso de este se traerá una imagen aleatoria entre las cuales el bot dispone, y para mantenerlo activo fue levantado a la plataforma de Heroku.
